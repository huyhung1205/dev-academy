Bài 3

Biến, hằng và kiểu dữ liệu

Bài học này giúp bạn biết cách lưu thông tin vào biến, dùng hằng khi dữ liệu không đổi, và chọn kiểu dữ liệu phù hợp để tránh lỗi ngay từ đầu.

1. Giới thiệu

Khái niệm

Biến là vùng nhớ đặt tên; hằng là giá trị không thay đổi; kiểu dữ liệu cho biết dữ liệu đó thuộc dạng gì.

Mục tiêu bài học

Khai báo đúng, gán đúng, hiểu phạm vi giá trị và chọn kiểu dữ liệu phù hợp.

Kiến thức tiền đề

Biết in ra màn hình và hiểu chương trình C++ bắt đầu từ main().

Video mô phỏng sẽ được cập nhật sau.

2. Tài liệu lý thuyết

2.1 Biến

Khái niệm: Biến là nơi lưu trữ một giá trị có thể thay đổi trong chương trình.

Tại sao cần học: Hầu như mọi bài toán đều phải lưu dữ liệu tạm thời như tuổi, điểm, số lượng, tên.

Cú pháp:

int age = 20;
double score = 8.5;
string name = "An";

Cách hoạt động: Khi khai báo, chương trình cấp vùng nhớ cho biến; khi gán giá trị, dữ liệu được lưu vào vùng nhớ đó.

Lỗi thường gặp: Dùng biến trước khi gán giá trị, đặt tên khó hiểu, hoặc chọn kiểu dữ liệu quá nhỏ.

2.2 Hằng

Khái niệm: Hằng là giá trị không thay đổi trong suốt thời gian chạy chương trình.

Tại sao cần học: Các giá trị như số ngày trong tuần, số pi hoặc giới hạn điểm tối đa nên để dạng hằng để tránh sửa nhầm.

Cú pháp:

const double PI = 3.14159;
const int MAX_STUDENT = 50;

Cách hoạt động: const nói với compiler rằng giá trị này chỉ đọc, không được gán lại.

Lỗi thường gặp: Cố gắng thay đổi hằng sau khi khai báo hoặc quên dùng const cho giá trị cố định.

2.3 Kiểu dữ liệu

Khái niệm: Kiểu dữ liệu cho biết biến lưu số nguyên, số thực, ký tự hay chuỗi.

Tại sao cần học: Chọn đúng kiểu giúp chương trình chính xác hơn và tiết kiệm bộ nhớ hơn.

Các kiểu cơ bản: int cho số nguyên, double cho số thực, char cho một ký tự, string cho chuỗi ký tự.

Lỗi thường gặp: Dùng int để lưu điểm 8.5 hoặc dùng char khi cần tên đầy đủ.

2.4 Phạm vi giá trị và kiểu phù hợp

Khái niệm: Mỗi kiểu dữ liệu có phạm vi và cách lưu khác nhau.

Tại sao cần học: Nếu chọn sai kiểu, dữ liệu có thể bị mất phần thập phân hoặc bị tràn số.

Minh họa: Tuổi nên dùng int, điểm trung bình nên dùng double, tên sinh viên nên dùng string.

Lỗi thường gặp: Dùng kiểu không phù hợp chỉ vì “code vẫn chạy”, trong khi kết quả đã sai từ đầu.

3. Ví dụ minh họa

Ví dụ cơ bản: Khai báo biến và hằng

#include <iostream>
using namespace std;

int main() {
    // Khai báo biến tuổi và điểm
    int age = 18;
    double score = 8.75;

    // Khai báo hằng số pi
    const double PI = 3.14159;

    // In ra các giá trị vừa khai báo
    cout << "Tuoi: " << age << endl;
    cout << "Diem: " << score << endl;
    cout << "PI: " << PI << endl;

    return 0;
}

Chương trình làm gì: Lưu và in ra biến, hằng, số nguyên và số thực.

Luồng thực thi: Khai báo biến, gán giá trị, in từng giá trị theo thứ tự.

Kết quả đầu ra: Hiển thị tuổi, điểm và giá trị PI.

Ví dụ thực tế: Tính diện tích hình tròn

#include <iostream>
using namespace std;

int main() {
    // Khai báo hằng số pi để dùng trong công thức
    const double PI = 3.14159;

    // Khai báo bán kính và diện tích
    double radius, area;

    // Nhập bán kính từ người dùng
    cout << "Nhap ban kinh: ";
    cin >> radius;

    // Tính diện tích hình tròn
    area = PI * radius * radius;

    // In kết quả
    cout << "Dien tich hinh tron la: " << area << endl;

    return 0;
}

Chương trình làm gì: Nhập bán kính và tính diện tích hình tròn.

Luồng thực thi: Nhập radius, dùng hằng PI, tính công thức, rồi in kết quả.

Kết quả đầu ra: Nếu bán kính là 2 thì diện tích xấp xỉ 12.56636.

6. Câu hỏi tự luận tự ôn tập

Câu 1. Vì sao sinh viên mới học C++ cần phân biệt biến và hằng?

Gợi ý: biến dùng cho dữ liệu thay đổi, hằng dùng cho dữ liệu cố định. Phân biệt rõ giúp code dễ đọc, tránh sửa nhầm giá trị và làm chương trình đúng hơn.

Câu 2. Hãy nêu ví dụ thực tế cho int, double và string.

Gợi ý: int cho số lượng sinh viên, double cho điểm trung bình, string cho họ tên. Chọn đúng kiểu giúp dữ liệu lưu trữ và tính toán chính xác.
