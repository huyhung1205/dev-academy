Nhập xuất dữ liệu

Bài học này dạy bạn cách nhận dữ liệu từ bàn phím, hiển thị kết quả rõ ràng và tránh các lỗi rất hay gặp khi mới học.

1. Giới thiệu

Khái niệm

Nhập xuất là cách chương trình nhận dữ liệu từ người dùng và in kết quả ra màn hình.

Mục tiêu

Biết dùng `cin`, `cout`, `getline` và hiểu khi nào cần xóa bộ đệm nhập.

Tiền đề

Đã biết biến, kiểu dữ liệu và biểu thức cơ bản từ các bài trước.

Video mô phỏng

Video mô phỏng sẽ được cập nhật sau.

2. Tài liệu lý thuyết

2.1 `cout` và cách xuất dữ liệu

Khái niệm: `cout` dùng để in dữ liệu ra màn hình.

Tại sao cần học: Không có xuất dữ liệu thì người dùng không biết chương trình đã tính ra gì.

Cú pháp:

cout << "Xin chao" << endl;

Cách hoạt động: Dấu `<<` đưa dữ liệu sang luồng xuất. `endl` xuống dòng và làm sạch bộ đệm.

Lỗi thường gặp: Quên `endl` khi muốn xuống dòng hoặc ghép chuỗi với số sai thứ tự.

2.2 `cin` và cách nhập dữ liệu

Khái niệm: `cin` nhận dữ liệu từ bàn phím và lưu vào biến.

Tại sao cần học: Hầu hết chương trình thực tế đều cần người dùng nhập thông tin.

Cú pháp:

int age;
cin >> age;

Cách hoạt động: Dấu `>>` lấy dữ liệu từ bàn phím và đưa vào biến.

Lỗi thường gặp: Nhập sai kiểu dữ liệu hoặc quên kiểm tra dữ liệu đầu vào.

2.3 `getline` và nhập chuỗi có khoảng trắng

Khái niệm: `getline` đọc cả một dòng văn bản, kể cả dấu cách.

Tại sao cần học: Tên người, địa chỉ hay ghi chú thường có nhiều từ.

Cách hoạt động: `getline(cin, name);` đọc đến khi gặp ký tự xuống dòng.

Lỗi thường gặp: Dùng `cin` cho họ tên nên chỉ lấy được một từ đầu tiên.

2.4 Định dạng và lỗi nhập

Khái niệm: Định dạng là cách in dữ liệu dễ nhìn; lỗi nhập là khi người dùng nhập sai kiểu.

Tại sao cần học: Chương trình thân thiện phải hướng dẫn người dùng và in kết quả rõ ràng.

Lỗi thường gặp: Sau khi dùng `cin` rồi đến `getline`, quên xử lý ký tự xuống dòng còn dư.

3. Ví dụ minh họa

Ví dụ cơ bản: Nhập tên và tuổi

#include <iostream>
using namespace std;

int main() {
    // Khai báo biến lưu tên và tuổi
    string name;
    int age;

    // Nhập họ tên bằng getline để lấy được cả dấu cách
    cout << "Nhap ho ten: ";
    getline(cin, name);

    // Nhập tuổi bằng cin
    cout << "Nhap tuoi: ";
    cin >> age;

    // In lại thông tin
    cout << "Xin chao " << name << ", ban " << age << " tuoi." << endl;
    return 0;
}

Chương trình làm gì: Nhập tên và tuổi rồi chào người dùng.

Luồng thực thi: Nhận chuỗi bằng `getline`, nhận số bằng `cin`, sau đó in kết quả.

Kết quả đầu ra: Một lời chào có tên và tuổi của người dùng.

Ví dụ thực tế: Tính tổng hai số thực

#include <iostream>
using namespace std;

int main() {
    // Khai báo hai số thực
    double a, b;

    // Nhập dữ liệu từ bàn phím
    cout << "Nhap so thu nhat: ";
    cin >> a;
    cout << "Nhap so thu hai: ";
    cin >> b;

    // Tính tổng
    double sum = a + b;

    // In kết quả có định dạng rõ ràng
    cout << "Tong cua hai so la: " << sum << endl;
    return 0;
}

Chương trình làm gì: Cộng hai số thực do người dùng nhập.

Luồng thực thi: Nhập `a`, nhập `b`, tính tổng, in ra màn hình.

Kết quả đầu ra: Ví dụ nhập 3.5 và 2.5 thì kết quả là 6.

6. Câu hỏi tự luận tự ôn tập

Câu 1. Vì sao `getline` hữu ích khi nhập dữ liệu?

Gợi ý: `getline` đọc được cả chuỗi có dấu cách, nên phù hợp cho họ tên, địa chỉ, mô tả dài và nhiều trường hợp nhập văn bản thực tế.

Câu 2. Nêu một lỗi hay gặp khi dùng `cin` và `getline` liên tiếp.

Gợi ý: sau `cin` thường còn ký tự xuống dòng trong bộ đệm, khiến `getline` đọc ngay dòng trống. Có thể xử lý bằng cách dùng `cin.ignore()` trước `getline`.
