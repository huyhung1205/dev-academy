Câu lệnh điều kiện

Bài học này giúp bạn cho chương trình biết phải chọn nhánh nào khi điều kiện đúng hoặc sai.

1. Giới thiệu

Khái niệm

Điều kiện là câu hỏi đúng/sai để chương trình quyết định sẽ đi theo nhánh nào.

Mục tiêu

Dùng thành thạo `if`, `if...else`, `else if` và `switch` ở mức cơ bản.

Tiền đề

Cần hiểu toán tử so sánh và toán tử logic từ bài 4.

2. Tài liệu lý thuyết

2.1 `if` và `if...else`

Khái niệm: `if` dùng để kiểm tra điều kiện; `else` là nhánh còn lại khi điều kiện sai.

Tại sao cần học: Rất nhiều bài toán cần phân loại như đậu/rớt, chẵn/lẻ, lớn/nhỏ.

Cú pháp:

if (dieu_kien) {
    // nhánh đúng
} else {
    // nhánh sai
}

Lỗi thường gặp: Không dùng ngoặc đúng chỗ, viết nhầm `=` thành `==`.

2.2 `else if`

Khái niệm: Dùng khi có nhiều mức phân loại.

Tại sao cần học: Một kết quả thực tế thường không chỉ có 2 trạng thái mà có nhiều mức như giỏi, khá, trung bình, yếu.

Cách hoạt động: Chương trình kiểm tra lần lượt từ trên xuống dưới, gặp điều kiện đúng đầu tiên thì chạy nhánh đó.

Lỗi thường gặp: Sắp xếp điều kiện sai thứ tự khiến kết quả bị lệch.

2.3 `switch`

Khái niệm: `switch` phù hợp khi kiểm tra một biến với nhiều giá trị cụ thể.

Tại sao cần học: Menu chương trình, ngày trong tuần hay mã lựa chọn thường dùng `switch`.

Cách hoạt động: Chọn `case` khớp với giá trị hiện tại, rồi dùng `break` để thoát.

Lỗi thường gặp: Quên `break` làm chương trình rơi sang các case phía dưới.

2.4 Điều kiện lồng nhau

Khái niệm: Một điều kiện có thể nằm trong điều kiện khác.

Tại sao cần học: Ví dụ kiểm tra học bổng có thể cần xét cả điểm trung bình lẫn chuyên cần.

Lỗi thường gặp: Viết lồng quá sâu làm code khó đọc, nên tách lại bằng hàm hoặc gom điều kiện hợp lý.

3. Ví dụ minh họa

Ví dụ cơ bản: Kiểm tra chẵn lẻ

#include <iostream>
using namespace std;

int main() {
    // Khai báo số nguyên cần kiểm tra
    int n;

    // Nhập số từ bàn phím
    cout << "Nhap mot so nguyen: ";
    cin >> n;

    // Kiểm tra chẵn hay lẻ
    if (n % 2 == 0) {
        cout << n << " la so chan." << endl;
    } else {
        cout << n << " la so le." << endl;
    }

    return 0;
}

Chương trình làm gì: Kiểm tra một số nguyên là chẵn hay lẻ.

Luồng thực thi: Nhập `n`, lấy phần dư khi chia cho 2, rồi chọn nhánh phù hợp.

Kết quả đầu ra: In ra “số chẵn” hoặc “số lẻ”.

Ví dụ thực tế: Xếp loại kết quả học tập

#include <iostream>
using namespace std;

int main() {
    // Khai báo điểm trung bình
    double average;

    // Nhập điểm
    cout << "Nhap diem trung binh: ";
    cin >> average;

    // Xếp loại theo nhiều mức
    if (average >= 8.5) {
        cout << "Xep loai: Gioi" << endl;
    } else if (average >= 7.0) {
        cout << "Xep loai: Kha" << endl;
    } else if (average >= 5.0) {
        cout << "Xep loai: Trung binh" << endl;
    } else {
        cout << "Xep loai: Yeu" << endl;
    }

    return 0;
}

Chương trình làm gì: Xếp loại học tập theo điểm trung bình.

Luồng thực thi: Nhập điểm, so sánh lần lượt từ mức cao đến thấp, in ra kết quả đầu tiên phù hợp.

Kết quả đầu ra: Ví dụ nhập 7.8 thì chương trình in “Kha”.

5. Trắc nghiệm

1. `if` dùng để làm gì?

2. `switch` phù hợp khi nào?

3. Vì sao cần `break` trong `switch`?

4. `else if` dùng khi nào?

5. Toán tử `%` thường dùng trong bài nào?

6. Câu hỏi tự luận tự ôn tập

Câu 1. Hãy giải thích vì sao cần sắp xếp các nhánh `else if` từ lớn đến nhỏ hoặc từ cụ thể đến tổng quát.

Gợi ý: điều kiện được kiểm tra từ trên xuống. Nếu đặt điều kiện rộng trước, các nhánh hẹp hơn có thể không bao giờ chạy.

Câu 2. Khi nào nên dùng `switch` thay vì `if...else`?

Gợi ý: dùng `switch` khi cần xử lý một biến với nhiều giá trị rời rạc, ví dụ menu lựa chọn. `if...else` phù hợp hơn khi điều kiện phức tạp.
