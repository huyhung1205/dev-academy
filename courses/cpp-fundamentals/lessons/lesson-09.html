Giới thiệu về con trỏ

Con trỏ là nền tảng rất quan trọng để hiểu bộ nhớ, địa chỉ và cách nhiều cấu trúc dữ liệu hoạt động trong C++.

1. Giới thiệu

Khái niệm

Con trỏ là biến lưu địa chỉ bộ nhớ của một biến khác.

Mục tiêu

Biết khai báo, lấy địa chỉ, truy cập gián tiếp và hiểu ý nghĩa của `nullptr`.

Tiền đề

Đã biết biến, hàm và phạm vi biến.

Video mô phỏng

Video mô phỏng sẽ được cập nhật sau.

2. Tài liệu lý thuyết

2.1 Khái niệm địa chỉ và con trỏ

Khái niệm: Mỗi biến nằm ở một vị trí trong bộ nhớ, vị trí đó gọi là địa chỉ.

Tại sao cần học: Khi hiểu địa chỉ, bạn mới hiểu vì sao con trỏ có thể làm việc trực tiếp với bộ nhớ.

Cú pháp:

int x = 10;
int* p = &x;

Cách hoạt động: `&x` lấy địa chỉ của `x`, còn `p` lưu địa chỉ đó. Dấu `*p` dùng để truy cập giá trị tại địa chỉ.

Lỗi thường gặp: Quên khởi tạo con trỏ hoặc dùng con trỏ chưa hợp lệ.

2.2 Truy cập gián tiếp

Khái niệm: Khi dùng `*p`, chương trình đọc hoặc ghi vào vùng nhớ mà `p` đang trỏ tới.

Tại sao cần học: Đây là cơ chế nền tảng của tham chiếu hàm, mảng động và cấu trúc dữ liệu liên kết.

Lỗi thường gặp: Nhầm giữa `*` trong khai báo và `*` khi truy cập giá trị.

2.3 `nullptr`

Khái niệm: `nullptr` là giá trị con trỏ không trỏ đến đâu cả.

Tại sao cần học: Nó giúp báo rằng con trỏ đang rỗng, tránh truy cập bừa vào vùng nhớ không hợp lệ.

Lỗi thường gặp: Quên kiểm tra con trỏ trước khi dùng.

2.4 Tại sao con trỏ quan trọng

Khái niệm: Con trỏ cho phép thao tác với địa chỉ và truyền dữ liệu hiệu quả hơn trong một số tình huống.

Tại sao cần học: Đây là bước đệm quan trọng để học mảng, chuỗi, cấp phát động, danh sách liên kết và cây.

3. Ví dụ minh họa

Ví dụ cơ bản: Lấy địa chỉ và giá trị

#include <iostream>
using namespace std;

int main() {
    // Khai báo biến số nguyên
    int x = 25;

    // Khai báo con trỏ trỏ tới x
    int* p = &x;

    // In giá trị và địa chỉ
    cout << "Gia tri x: " << x << endl;
    cout << "Dia chi cua x: " << p << endl;
    cout << "Gia tri qua con tro: " << *p << endl;

    return 0;
}

Chương trình làm gì: Minh họa con trỏ lưu địa chỉ của biến.

Luồng thực thi: Tạo biến `x`, gán địa chỉ cho `p`, in giá trị trực tiếp và gián tiếp.

Kết quả đầu ra: In ra giá trị, địa chỉ và giá trị thông qua con trỏ.

Ví dụ thực tế: Hoán đổi hai số bằng con trỏ

#include <iostream>
using namespace std;

// Hàm hoán đổi hai số qua địa chỉ
void swapNumbers(int* a, int* b) {
    // Dùng biến tạm để đổi giá trị
    int temp = *a;
    *a = *b;
    *b = temp;
}

int main() {
    // Khai báo hai số nguyên
    int x = 3, y = 8;

    // Gọi hàm hoán đổi bằng địa chỉ
    swapNumbers(&x, &y);

    // In kết quả sau khi đổi
    cout << "x = " << x << ", y = " << y << endl;
    return 0;
}

Chương trình làm gì: Đổi chỗ hai giá trị thông qua địa chỉ.

Luồng thực thi: Truyền địa chỉ vào hàm, hàm thay đổi giá trị gốc, rồi `main()` in ra kết quả.

Kết quả đầu ra: `x = 8, y = 3`.

6. Câu hỏi tự luận tự ôn tập

Câu 1. Vì sao con trỏ được xem là bài học nền tảng cho cấu trúc dữ liệu?

Gợi ý: con trỏ giúp hiểu địa chỉ, thao tác gián tiếp với dữ liệu và là cơ sở của danh sách liên kết, cây, ngăn xếp động và nhiều cấu trúc khác.

Câu 2. Hãy giải thích sự khác nhau giữa `&x` và `*p`.

Gợi ý: `&x` lấy địa chỉ của biến x, còn `*p` truy cập giá trị tại địa chỉ mà con trỏ p đang giữ. Một bên là lấy địa chỉ, một bên là đọc/ghi giá trị.
