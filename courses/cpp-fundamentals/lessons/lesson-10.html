Bài 10

Mảng một chiều

Mảng một chiều giúp lưu nhiều giá trị cùng kiểu trong một biến duy nhất, rất hay dùng khi xử lý danh sách dữ liệu.

1. Giới thiệu

Khái niệm

Mảng là tập hợp nhiều phần tử cùng kiểu, đặt liên tiếp trong bộ nhớ.

Mục tiêu

Khai báo, nhập, duyệt và xử lý mảng một chiều bằng vòng lặp.

Tiền đề

Đã biết vòng lặp, hàm và con trỏ cơ bản.

Video mô phỏng

Video mô phỏng sẽ được cập nhật sau.

2. Tài liệu lý thuyết

2.1 Khái niệm mảng

Khái niệm: Mảng là tập hợp phần tử cùng kiểu dữ liệu, có chỉ số bắt đầu từ 0.

Tại sao cần học: Khi cần lưu nhiều điểm số, tên sản phẩm hoặc số lượng, mảng gọn hơn nhiều biến rời rạc.

Cú pháp:

int numbers[5] = {1, 2, 3, 4, 5};

Cách hoạt động: `numbers[0]` là phần tử đầu tiên, `numbers[4]` là phần tử cuối.

Lỗi thường gặp: Truy cập vượt quá chỉ số hợp lệ.

2.2 Duyệt mảng

Khái niệm: Duyệt mảng là đi qua từng phần tử để xử lý.

Tại sao cần học: Rất nhiều bài toán như tính tổng, tìm max, đếm phần tử đều cần duyệt mảng.

Lỗi thường gặp: Dùng sai điều kiện `i < n` và `i <= n`.

2.3 Các thao tác cơ bản

Khái niệm: Nhập, in, tìm kiếm, cập nhật và sắp xếp là những thao tác thường gặp.

Tại sao cần học: Đây là nền tảng để học cấu trúc dữ liệu sâu hơn sau này.

2.4 Lỗi thường gặp

Lỗi thường gặp: Quên khai báo kích thước hợp lý, nhập sai số phần tử hoặc dùng mảng khi cần cấu trúc dữ liệu linh hoạt hơn.

3. Ví dụ minh họa

Ví dụ cơ bản: Nhập và in mảng

#include <iostream>
using namespace std;

int main() {
    // Khai báo mảng 5 phần tử
    int a[5];

    // Nhập từng phần tử
    for (int i = 0; i < 5; i++) {
        cout << "Nhap a[" << i << "]: ";
        cin >> a[i];
    }

    // In lại mảng
    cout << "Mang vua nhap: ";
    for (int i = 0; i < 5; i++) {
        cout << a[i] << " ";
    }
    cout << endl;
    return 0;
}

Chương trình làm gì: Nhập 5 số nguyên và in lại chúng.

Luồng thực thi: Duyệt mảng để nhập, rồi duyệt lại để in.

Kết quả đầu ra: Hiển thị 5 số theo đúng thứ tự nhập.

Ví dụ thực tế: Tìm số lớn nhất trong mảng

#include <iostream>
using namespace std;

int main() {
    // Mảng điểm và biến lưu số lượng phần tử
    int scores[5];
    int maxScore;

    // Nhập dữ liệu
    for (int i = 0; i < 5; i++) {
        cout << "Nhap diem thu " << i + 1 << ": ";
        cin >> scores[i];
    }

    // Giả sử phần tử đầu tiên là lớn nhất
    maxScore = scores[0];

    // So sánh với các phần tử còn lại
    for (int i = 1; i < 5; i++) {
        if (scores[i] > maxScore) {
            maxScore = scores[i];
        }
    }

    // In kết quả
    cout << "Diem lon nhat la: " << maxScore << endl;
    return 0;
}

Chương trình làm gì: Tìm điểm lớn nhất trong 5 giá trị.

Luồng thực thi: Nhập mảng, gán phần tử đầu làm max, duyệt các phần tử còn lại để cập nhật.

Kết quả đầu ra: In ra giá trị lớn nhất của mảng.

2. Mảng dùng để làm gì?

3. `a[i]` nghĩa là gì?

6. Câu hỏi tự luận tự ôn tập

Câu 1. Vì sao mảng rất hữu ích khi lưu danh sách dữ liệu?

Gợi ý: mảng gom nhiều giá trị cùng kiểu vào một nơi, giúp duyệt bằng vòng lặp và xử lý dữ liệu danh sách thuận tiện hơn nhiều biến rời rạc.

Câu 2. Hãy giải thích lỗi vượt chỉ số mảng bằng ví dụ đơn giản.

Gợi ý: nếu mảng có 5 phần tử thì chỉ số hợp lệ là 0 đến 4. Truy cập `a[5]` là sai vì đi ra ngoài vùng nhớ đã cấp cho mảng.
