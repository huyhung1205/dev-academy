Bài 11

Mảng nhiều chiều

Mảng nhiều chiều giúp lưu dữ liệu theo hàng và cột, rất phù hợp với bảng điểm, ma trận và dữ liệu dạng lưới.

1. Giới thiệu

Khái niệm

Mảng nhiều chiều là mảng của mảng, phổ biến nhất là mảng 2 chiều.

Mục tiêu

Khai báo, nhập, in và duyệt mảng 2 chiều bằng vòng lặp lồng nhau.

Tiền đề

Cần nắm mảng một chiều và vòng lặp.

Video mô phỏng

Video mô phỏng sẽ được cập nhật sau.

2. Tài liệu lý thuyết

2.1 Khái niệm

Khái niệm: Mảng 2 chiều lưu dữ liệu theo dạng bảng, gồm hàng và cột.

Tại sao cần học: Bảng điểm, danh sách phòng học, bàn cờ và ma trận đều là ví dụ quen thuộc.

Cú pháp:

int a[2][3] = {{1, 2, 3}, {4, 5, 6}};

Cách hoạt động: a[i][j] truy cập phần tử ở hàng i, cột j.

Lỗi thường gặp: Nhầm chỉ số hàng và cột hoặc truy cập vượt giới hạn.

2.2 Duyệt mảng 2 chiều

Khái niệm: Duyệt mảng 2 chiều thường dùng 2 vòng lặp lồng nhau.

Tại sao cần học: Đây là cách chuẩn để in hoặc tính toán trên toàn bộ bảng dữ liệu.

2.3 Ứng dụng

Ứng dụng: Lưu điểm theo nhiều môn, xử lý ma trận trong toán học, duyệt bảng dữ liệu nhỏ.

2.4 Lỗi thường gặp

Lỗi thường gặp: Dùng sai số cột/hàng, quên khởi tạo hoặc viết vòng lặp sai điều kiện.

3. Ví dụ minh họa

Ví dụ cơ bản: Nhập và in ma trận 2x3

#include <iostream>
using namespace std;

int main() {
    // Khai báo ma trận 2 hàng 3 cột
    int a[2][3];

    // Nhập từng phần tử
    for (int i = 0; i < 2; i++) {
        for (int j = 0; j < 3; j++) {
            cout << "Nhap a[" << i << "][" << j << "]: ";
            cin >> a[i][j];
        }
    }

    // In ma trận
    cout << "Ma tran vua nhap:" << endl;
    for (int i = 0; i < 2; i++) {
        for (int j = 0; j < 3; j++) {
            cout << a[i][j] << " ";
        }
        cout << endl;
    }
    return 0;
}

Chương trình làm gì: Nhập một ma trận 2x3 và in lại.

Luồng thực thi: Duyệt theo hàng và cột để nhập, sau đó duyệt lại để xuất.

Kết quả đầu ra: Hiển thị bảng dữ liệu đúng thứ tự đã nhập.

Ví dụ thực tế: Tính tổng từng hàng

#include <iostream>
using namespace std;

int main() {
    // Ma trận điểm 2x3
    int scores[2][3] = {{8, 7, 9}, {6, 5, 8}};

    // Duyệt từng hàng
    for (int i = 0; i < 2; i++) {
        int rowSum = 0;

        // Cộng từng phần tử trong hàng
        for (int j = 0; j < 3; j++) {
            rowSum += scores[i][j];
        }

        // In tổng của hàng hiện tại
        cout << "Tong hang " << i + 1 << ": " << rowSum << endl;
    }
    return 0;
}

Chương trình làm gì: Tính tổng từng hàng trong ma trận điểm.

Luồng thực thi: Lặp qua từng hàng, cộng các phần tử trong hàng, rồi in ra kết quả.

Kết quả đầu ra: Hiển thị tổng của từng hàng.

6. Câu hỏi tự luận tự ôn tập

Câu 1. Vì sao mảng 2 chiều giống một bảng dữ liệu?

Gợi ý: vì dữ liệu được chia thành hàng và cột, nên rất trực quan khi biểu diễn các bảng điểm, ma trận hay danh sách theo dòng.

Câu 2. Hãy mô tả cách duyệt một ma trận 2 chiều.

Gợi ý: dùng vòng lặp ngoài để chạy qua từng hàng và vòng lặp trong để chạy qua từng cột của hàng đó, xử lý từng phần tử một cách tuần tự.
