Chuỗi ký tự

Chuỗi giúp lưu và xử lý văn bản như tên người, địa chỉ, mô tả, thông báo hay dữ liệu nhập từ bàn phím.

1. Giới thiệu

Khái niệm

Chuỗi là dãy ký tự liên tiếp, thường được lưu bằng string hoặc mảng ký tự.

Mục tiêu

Biết tạo, nối, so sánh và nhập chuỗi đúng cách.

Tiền đề

Đã biết biến, nhập xuất và vòng lặp cơ bản.

Video mô phỏng

Video mô phỏng sẽ được cập nhật sau.

2. Tài liệu lý thuyết

2.1 string

Khái niệm: string là kiểu dữ liệu trong C++ dùng để lưu chuỗi văn bản.

Tại sao cần học: Gần như mọi chương trình đều cần xử lý tên, mã, thông báo hoặc nội dung người dùng nhập.

Cú pháp:

string name = "An";

Cách hoạt động: string quản lý chuỗi tự động, dễ dùng hơn mảng ký tự thủ công.

Lỗi thường gặp: Dùng cin để nhập tên có dấu cách nên chỉ lấy một từ.

2.2 Nối và truy cập chuỗi

Khái niệm: Có thể ghép chuỗi bằng + và truy cập từng ký tự theo chỉ số.

Tại sao cần học: Rất hữu ích khi tạo thông báo hoặc phân tích dữ liệu văn bản.

2.3 So sánh chuỗi

Khái niệm: Chuỗi có thể so sánh bằng toán tử so sánh để kiểm tra bằng nhau hoặc thứ tự.

Tại sao cần học: Dùng trong đăng nhập, kiểm tra mật khẩu, tìm kiếm tên và lọc dữ liệu.

2.4 Lỗi thường gặp

Lỗi thường gặp: Nhầm giữa ký tự đơn char và chuỗi string, hoặc quên xử lý khoảng trắng khi nhập.

3. Ví dụ minh họa

Ví dụ cơ bản: Nhập và in tên

#include <iostream>
using namespace std;

int main() {
    // Khai báo biến lưu họ tên
    string name;

    // Nhập họ tên đầy đủ
    cout << "Nhap ho ten: ";
    getline(cin, name);

    // In lời chào
    cout << "Xin chao, " << name << "!" << endl;
    return 0;
}

Chương trình làm gì: Nhập và chào theo họ tên người dùng.

Luồng thực thi: getline lấy cả dòng, sau đó cout in lời chào.

Kết quả đầu ra: Chào đúng tên có dấu cách.

Ví dụ thực tế: Kiểm tra mã sinh viên

#include <iostream>
using namespace std;

int main() {
    // Khai báo mã sinh viên và mã chuẩn
    string studentId;
    string correctId = "IT2026";

    // Nhập mã sinh viên
    cout << "Nhap ma sinh vien: ";
    getline(cin, studentId);

    // So sánh chuỗi
    if (studentId == correctId) {
        cout << "Ma hop le." << endl;
    } else {
        cout << "Ma khong hop le." << endl;
    }
    return 0;
}

Chương trình làm gì: So sánh mã sinh viên với mã mẫu.

Luồng thực thi: Nhập chuỗi, so sánh với chuỗi chuẩn, in kết quả đúng hoặc sai.

Kết quả đầu ra: Báo mã hợp lệ hay không.

6. Câu hỏi tự luận tự ôn tập

Câu 1. Vì sao string thường dễ dùng hơn mảng ký tự?

Gợi ý: string tự quản lý độ dài và thao tác ghép chuỗi, phù hợp cho người mới hơn so với mảng ký tự thủ công vốn dễ lỗi hơn.

Câu 2. Hãy nêu một ví dụ thực tế cần nhập chuỗi có dấu cách.

Gợi ý: họ tên, địa chỉ, mô tả sản phẩm hoặc lời nhắn đều có thể chứa khoảng trắng, nên getline rất phù hợp.
