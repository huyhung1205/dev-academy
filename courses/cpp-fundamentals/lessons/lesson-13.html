Bài 13

Kiểu dữ liệu cấu trúc (struct)

struct giúp gom nhiều thuộc tính liên quan vào một kiểu dữ liệu duy nhất, rất phù hợp để mô hình hóa thông tin thực tế.

1. Giới thiệu

Khái niệm

Struct là kiểu dữ liệu do người dùng tự định nghĩa, chứa nhiều trường dữ liệu khác nhau.

Mục tiêu

Biết khai báo struct, tạo biến struct và truy cập thành viên bằng dấu chấm.

Tiền đề

Đã biết kiểu dữ liệu, mảng và hàm.

Video mô phỏng

Video mô phỏng sẽ được cập nhật sau.

2. Tài liệu lý thuyết

2.1 Khái niệm struct

Khái niệm: Struct là một khuôn mẫu dùng để gom nhiều thông tin liên quan vào cùng một đối tượng.

Tại sao cần học: Thay vì lưu tên, tuổi, điểm thành nhiều biến riêng rẽ, ta có thể gộp thành một sinh viên duy nhất.

Cú pháp:

struct Student {
    string name;
    int age;
    double gpa;
};

Cách hoạt động: Khi tạo biến từ struct, mỗi trường trong struct đều có vùng nhớ riêng.

Lỗi thường gặp: Quên dấu ; sau khi kết thúc struct hoặc dùng sai tên trường.

2.2 Truy cập thành viên

Khái niệm: Dùng dấu chấm . để truy cập thành viên của một biến struct.

Tại sao cần học: Đây là cách đọc và gán dữ liệu cho đối tượng đã mô hình hóa.

2.3 Struct trong mảng

Khái niệm: Có thể tạo mảng struct để lưu nhiều sinh viên hoặc sản phẩm.

Tại sao cần học: Đây là cách rất phổ biến trong quản lý dữ liệu thực tế.

2.4 Lỗi thường gặp

Lỗi thường gặp: Trộn struct với class, quên khởi tạo dữ liệu hoặc dùng tên trường không nhất quán.

3. Ví dụ minh họa

Ví dụ cơ bản: Khai báo và in struct

#include <iostream>
using namespace std;

// Tạo kiểu dữ liệu sinh viên
struct Student {
    // Họ tên sinh viên
    string name;
    // Tuổi sinh viên
    int age;
    // Điểm trung bình
    double gpa;
};

int main() {
    // Khai báo một sinh viên
    Student sv;

    // Gán dữ liệu cho từng trường
    sv.name = "Nguyen Van A";
    sv.age = 19;
    sv.gpa = 8.2;

    // In thông tin sinh viên
    cout << sv.name << ", " << sv.age << ", " << sv.gpa << endl;
    return 0;
}

Chương trình làm gì: Lưu và in thông tin một sinh viên.

Luồng thực thi: Định nghĩa struct, tạo biến, gán dữ liệu, rồi in ra màn hình.

Kết quả đầu ra: Thông tin sinh viên hiển thị rõ ràng.

Ví dụ thực tế: Danh sách 2 sinh viên

#include <iostream>
using namespace std;

struct Student {
    // Tên sinh viên
    string name;
    // Điểm trung bình
    double gpa;
};

int main() {
    // Khai báo mảng 2 sinh viên
    Student classList[2];

    // Nhập dữ liệu cho từng sinh viên
    for (int i = 0; i < 2; i++) {
        cout << "Nhap ten sinh vien " << i + 1 << ": ";
        getline(cin, classList[i].name);
        cout << "Nhap GPA: ";
        cin >> classList[i].gpa;
        cin.ignore();
    }

    // In danh sách
    for (int i = 0; i < 2; i++) {
        cout << classList[i].name << " - " << classList[i].gpa << endl;
    }
    return 0;
}

Chương trình làm gì: Lưu và in danh sách sinh viên bằng mảng struct.

Luồng thực thi: Nhập từng sinh viên, lưu vào mảng, sau đó duyệt mảng để in.

Kết quả đầu ra: Danh sách 2 sinh viên với tên và GPA.

Câu 1. Vì sao struct phù hợp để lưu thông tin sinh viên?

Gợi ý: vì thông tin của sinh viên gồm nhiều trường liên quan như tên, tuổi, GPA. Struct gom chúng lại thành một đơn vị rõ ràng, dễ quản lý hơn.

Câu 2. Hãy mô tả cách dùng mảng struct trong quản lý danh sách lớp.

Gợi ý: tạo một struct cho sinh viên, sau đó tạo mảng các sinh viên. Mỗi phần tử trong mảng đại diện cho một người, rất thuận tiện cho việc duyệt, tìm kiếm và sắp xếp.
