Đọc ghi File

File giúp lưu dữ liệu lâu dài, tách dữ liệu ra khỏi chương trình và phục vụ các ứng dụng cần nhớ thông tin sau khi tắt máy.

1. Giới thiệu

Khái niệm

File là nơi lưu dữ liệu trên đĩa, chương trình có thể đọc vào hoặc ghi ra.

Mục tiêu

Biết mở, đọc, ghi và đóng file bằng `fstream`.

Tiền đề

Đã biết chuỗi, struct và hàm cơ bản.

Video mô phỏng

Video mô phỏng sẽ được cập nhật sau.

2. Tài liệu lý thuyết

2.1 Đọc và ghi file

Khái niệm: Đọc file là lấy dữ liệu từ file vào chương trình, ghi file là đưa dữ liệu từ chương trình ra file.

Tại sao cần học: Nếu chỉ lưu trong RAM, dữ liệu sẽ mất khi tắt chương trình.

Cú pháp:

#include <fstream>
ofstream outFile("data.txt");
ifstream inFile("data.txt");

Cách hoạt động: `ofstream` dùng để ghi, `ifstream` dùng để đọc, `fstream` có thể làm cả hai.

Lỗi thường gặp: Quên kiểm tra file mở thành công hoặc quên đóng file.

2.2 Ghi file văn bản

Khái niệm: Ghi văn bản là lưu chuỗi hoặc số theo dạng dễ đọc.

Tại sao cần học: Phù hợp cho danh sách sinh viên, báo cáo, nhật ký và dữ liệu text đơn giản.

2.3 Đọc file văn bản

Khái niệm: Đọc file giúp nạp lại dữ liệu đã lưu để sử dụng tiếp.

Tại sao cần học: Đây là bước nối giữa chương trình hiện tại và lần chạy sau.

2.4 Lỗi thường gặp

Lỗi thường gặp: Sai đường dẫn file, quên kiểm tra `is_open()`, hoặc không xử lý dữ liệu có khoảng trắng.

3. Ví dụ minh họa

Ví dụ cơ bản: Ghi một dòng vào file

#include <iostream>
 #include <fstream>
using namespace std;

int main() {
    // Mở file để ghi
    ofstream outFile("note.txt");

    // Kiểm tra file có mở được không
    if (!outFile.is_open()) {
        cout << "Khong mo duoc file." << endl;
        return 1;
    }

    // Ghi nội dung vào file
    outFile << "Xin chao file!";

    // Đóng file sau khi ghi xong
    outFile.close();
    cout << "Da ghi file thanh cong." << endl;
    return 0;
}

Chương trình làm gì: Ghi một dòng văn bản vào file.

Luồng thực thi: Mở file, kiểm tra, ghi chuỗi, đóng file, in thông báo.

Kết quả đầu ra: Tạo file `note.txt` chứa dòng chữ đã ghi.

Ví dụ thực tế: Lưu danh sách sinh viên

#include <iostream>
 #include <fstream>
using namespace std;

int main() {
    // Mở file để ghi
    ofstream outFile("students.txt");

    // Kiểm tra file
    if (!outFile.is_open()) {
        cout << "Khong mo duoc file." << endl;
        return 1;
    }

    // Ghi từng dòng thông tin sinh viên
    outFile << "Nguyen Van A, 8.5" << endl;
    outFile << "Tran Thi B, 9.0" << endl;

    // Đóng file
    outFile.close();
    cout << "Da luu danh sach sinh vien." << endl;
    return 0;
}

Chương trình làm gì: Lưu dữ liệu sinh viên vào file text.

Luồng thực thi: Mở file, ghi từng dòng, đóng file, báo hoàn tất.

Kết quả đầu ra: File `students.txt` có danh sách sinh viên.

6. Câu hỏi tự luận tự ôn tập

Câu 1. Vì sao file cần thiết trong chương trình thực tế?

Gợi ý: file giúp chương trình lưu trạng thái và dữ liệu giữa nhiều lần chạy, rất cần cho quản lý thông tin, báo cáo và nhập xuất dữ liệu lâu dài.

Gợi ý: đường dẫn sai, thiếu quyền truy cập hoặc quên kiểm tra file có mở thành công hay không đều là lỗi rất phổ biến.
