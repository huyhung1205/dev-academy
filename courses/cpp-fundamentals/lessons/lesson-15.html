Ôn tập tổng hợp và Mini Project

Bài cuối giúp bạn nối tất cả kiến thức nền tảng để tự làm một bài toán console nhỏ có cấu trúc rõ ràng.

1. Giới thiệu

Khái niệm

Mini project là bài thực hành ghép nhiều kiến thức đã học vào một chương trình hoàn chỉnh.

Mục tiêu

Biết thiết kế chương trình nhỏ có menu, dữ liệu, hàm và lưu trữ cơ bản.

Tiền đề

Cần nắm toàn bộ kiến thức từ bài 1 đến bài 14.

Video mô phỏng

Video mô phỏng sẽ được cập nhật sau.

2. Tài liệu lý thuyết

2.1 Tư duy tổng hợp

Khái niệm: Ôn tập tổng hợp là quá trình ghép biến, điều kiện, vòng lặp, hàm, mảng, struct và file thành một bài toán lớn hơn.

Tại sao cần học: Đây là cách kiểm tra xem bạn có thật sự hiểu nền tảng hay chưa.

2.2 Cách thiết kế mini project

Các bước: Xác định yêu cầu, chia chức năng nhỏ, chọn kiểu dữ liệu, viết hàm riêng và kiểm thử từng phần.

Lỗi thường gặp: Làm quá lớn ngay từ đầu hoặc không chia chức năng rõ ràng.

2.3 Gợi ý mini project

Gợi ý: Quản lý sinh viên console, quản lý thư viện nhỏ, hoặc quản lý sản phẩm đơn giản.

Tại sao cần học: Bài project giúp bạn làm quen với tư duy thiết kế phần mềm nhỏ, rất hữu ích khi học cấu trúc dữ liệu và giải thuật.

2.4 Lỗi thường gặp

Lỗi thường gặp: Không có menu rõ ràng, không có hàm tách riêng, dữ liệu nhập không được kiểm tra, và quên lưu file nếu cần.

3. Ví dụ minh họa

Ví dụ cơ bản: Menu lựa chọn

#include <iostream>
using namespace std;

int main() {
    // Hiển thị menu đơn giản
    int choice;

    cout << "1. Xem thong tin" << endl;
    cout << "2. Tinh tong" << endl;
    cout << "3. Thoat" << endl;
    cout << "Nhap lua chon: ";
    cin >> choice;

    // Xử lý theo lựa chọn
    if (choice == 1) {
        cout << "Ban chon xem thong tin." << endl;
    } else if (choice == 2) {
        cout << "Ban chon tinh tong." << endl;
    } else {
        cout << "Thoat chuong trinh." << endl;
    }
    return 0;
}

Chương trình làm gì: Minh họa menu đơn giản cho chương trình console.

Luồng thực thi: In menu, nhận lựa chọn, rẽ nhánh theo if...else.

Kết quả đầu ra: In thông báo tương ứng với lựa chọn.

Ví dụ thực tế: Mini project quản lý sinh viên

#include <iostream>
using namespace std;

// Cấu trúc lưu thông tin sinh viên
struct Student {
    string name;
    double gpa;
};

// Hàm nhập thông tin 1 sinh viên
void inputStudent(Student &s) {
    cout << "Nhap ten: ";
    getline(cin, s.name);
    cout << "Nhap GPA: ";
    cin >> s.gpa;
    cin.ignore();
}

// Hàm in thông tin 1 sinh viên
void printStudent(const Student &s) {
    cout << s.name << " - " << s.gpa << endl;
}

int main() {
    // Khai báo danh sách 2 sinh viên
    Student list[2];

    // Nhập dữ liệu
    for (int i = 0; i < 2; i++) {
        inputStudent(list[i]);
    }

    // In dữ liệu
    for (int i = 0; i < 2; i++) {
        printStudent(list[i]);
    }
    return 0;
}

Chương trình làm gì: Nhập và in danh sách sinh viên đơn giản.

Luồng thực thi: Dùng struct lưu dữ liệu, dùng hàm để nhập và in, rồi duyệt mảng sinh viên.

Kết quả đầu ra: Danh sách sinh viên được hiển thị lại sau khi nhập.

6. Câu hỏi tự luận tự ôn tập

Câu 1. Nếu tự làm mini project, bạn sẽ chia chương trình thành những phần nào?

Gợi ý: có thể chia thành phần nhập liệu, xử lý dữ liệu, hiển thị kết quả, và nếu cần thì thêm lưu file. Chia nhỏ như vậy giúp project dễ mở rộng và sửa lỗi hơn.

Câu 2. Điều gì là quan trọng nhất khi học xong khóa C++ cơ bản?

Gợi ý: điều quan trọng nhất là biết đọc đề, tách bài toán thành bước nhỏ, chọn cấu trúc dữ liệu phù hợp và tự viết chương trình hoàn chỉnh từ đầu đến cuối.
