Vector và mảng một chiều

Bài này đi từ mảng một chiều đến vector để bạn thấy cách lưu danh sách số liệu, duyệt, tìm max/min và đếm tần suất.

Mục tiêu bài học

  • Vector là mảng động của STL.
  • Mảng một chiều lưu liên tiếp trong bộ nhớ.
  • Chỉ số bắt đầu từ 0 và không có kiểm tra biên tự động.
  • Bài toán max/min, tổng, đếm xuất hiện.

Bài này đi từ mảng một chiều đến vector để bạn thấy cách lưu danh sách số liệu, duyệt, tìm max/min và đếm tần suất.

Trong bài này, điều quan trọng nhất là không chỉ nhớ định nghĩa mà còn phải biết bài toán nào nên dùng kỹ thuật nào, vì sao nó chạy nhanh hơn và khi nào không nên áp dụng. Khi đọc PDF gốc, hãy để ý các ví dụ và công thức vì chúng thường là điểm quyết định để làm bài tập.

Ở level của sinh viên năm nhất và người tự học, mục tiêu là hiểu thật rõ dữ liệu đang được lưu như thế nào, thao tác nào là O(1), thao tác nào là O(log n) hoặc O(n), rồi dùng kiến thức đó để chọn lời giải hợp lý hơn. Đây là tinh thần xuyên suốt toàn bộ khóa học.

Ví dụ minh họa

Ví dụ 1: Vector và thống kê điểm

#include <iostream>
#include <vector>
using namespace std;

int main() {
    int n;
    cin >> n;
    vector<int> a(n);
    for (int i = 0; i < n; i++) cin >> a[i];
    int mx = a[0], mn = a[0];
    long long sum = 0;
    for (int x : a) {
        sum += x;
        if (x > mx) mx = x;
        if (x < mn) mn = x;
    }
    cout << mx << '\n' << mn << '\n' << sum << endl;
    return 0;
}

Giải thích chương trình

  • Chương trình làm gì: Dùng vector để lưu danh sách điểm và thống kê nhanh.
  • Input: Nhập n và n điểm.
  • Output: In max, min, tổng.
  • Biến chính: Biến: a, mx, mn, sum.
  • Luồng chạy: Duyệt một lượt qua vector.
  • Độ phức tạp / kết quả kỳ vọng: O(n).

Ví dụ 2: Mảng một chiều và đếm số lần xuất hiện

#include <iostream>
using namespace std;

int main() {
    int n, x;
    cin >> n >> x;
    int a[1000];
    for (int i = 0; i < n; i++) cin >> a[i];
    int cnt = 0;
    for (int i = 0; i < n; i++) if (a[i] == x) cnt++;
    cout << cnt << endl;
    return 0;
}

Giải thích chương trình

  • Chương trình làm gì: Đếm số lần xuất hiện của x trong mảng.
  • Input: Nhập n, x và dãy số.
  • Output: In số lần xuất hiện của x.
  • Biến chính: Biến: a, x, cnt.
  • Luồng chạy: Duyệt toàn bộ mảng.
  • Độ phức tạp / kết quả kỳ vọng: O(n).
