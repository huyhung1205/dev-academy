Bài này đi qua các thuật toán sắp xếp cơ bản và tìm kiếm tuyến tính, nhị phân, cùng lower_bound và upper_bound.

Trong bài này, điều quan trọng nhất là không chỉ nhớ định nghĩa mà còn phải biết bài toán nào nên dùng kỹ thuật nào, vì sao nó chạy nhanh hơn và khi nào không nên áp dụng. Khi đọc PDF gốc, hãy để ý các ví dụ và công thức vì chúng thường là điểm quyết định để làm bài tập.

Ở level của sinh viên năm nhất và người tự học, mục tiêu là hiểu thật rõ dữ liệu đang được lưu như thế nào, thao tác nào là O(1), thao tác nào là O(log n) hoặc O(n), rồi dùng kiến thức đó để chọn lời giải hợp lý hơn. Đây là tinh thần xuyên suốt toàn bộ khóa học.

Ví dụ minh họa

Ví dụ 1: Selection sort

#include <iostream>
using namespace std;

int main() {
    int n;
    cin >> n;
    int a[1000];
    for (int i = 0; i < n; i++) cin >> a[i];
    for (int i = 0; i < n - 1; i++) {
        int pos = i;
        for (int j = i + 1; j < n; j++) if (a[j] < a[pos]) pos = j;
        swap(a[i], a[pos]);
    }
    for (int i = 0; i < n; i++) cout << a[i] << ' ';
    cout << endl;
    return 0;
}

Giải thích chương trình

  • Chương trình làm gì: Sắp xếp mảng bằng selection sort.
  • Input: Nhập n và dãy số.
  • Output: In dãy đã sắp xếp.
  • Biến chính: Biến: a, pos.
  • Luồng chạy: Mỗi lượt chọn phần tử nhỏ nhất còn lại.
  • Độ phức tạp / kết quả kỳ vọng: O(n^2).

Ví dụ 2: Binary search và lower_bound

#include <iostream>
#include <algorithm>
using namespace std;

int main() {
    int a[] = {1, 2, 2, 4, 5, 7};
    auto it1 = lower_bound(a, a + 6, 2);
    auto it2 = upper_bound(a, a + 6, 2);
    cout << (it1 - a) << '\n' << (it2 - a) << endl;
    return 0;
}

Giải thích chương trình

  • Chương trình làm gì: Dùng STL để tìm cận dưới và cận trên.
  • Input: Không cần input.
  • Output: In hai vị trí cận.
  • Biến chính: Biến: a, it1, it2.
  • Luồng chạy: Dùng thuật toán nhị phân của STL.
  • Độ phức tạp / kết quả kỳ vọng: O(log n).
