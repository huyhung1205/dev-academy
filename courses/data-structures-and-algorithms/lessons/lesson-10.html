Thuật toán sinh liệt kê có hệ thống các cấu hình như xâu nhị phân, tổ hợp, hoán vị, phân hoạch và tập con bằng bitmask.

Trong bài này, điều quan trọng nhất là không chỉ nhớ định nghĩa mà còn phải biết bài toán nào nên dùng kỹ thuật nào, vì sao nó chạy nhanh hơn và khi nào không nên áp dụng. Khi đọc PDF gốc, hãy để ý các ví dụ và công thức vì chúng thường là điểm quyết định để làm bài tập.

Ở level của sinh viên năm nhất và người tự học, mục tiêu là hiểu thật rõ dữ liệu đang được lưu như thế nào, thao tác nào là O(1), thao tác nào là O(log n) hoặc O(n), rồi dùng kiến thức đó để chọn lời giải hợp lý hơn. Đây là tinh thần xuyên suốt toàn bộ khóa học.

Ví dụ minh họa

Ví dụ 1: Sinh xâu nhị phân

#include <iostream>
using namespace std;

int main() {
    int n;
    cin >> n;
    for (int mask = 0; mask < (1 << n); mask++) {
        for (int i = n - 1; i >= 0; i--) cout << ((mask >> i) & 1);
        cout << endl;
    }
    return 0;
}

Giải thích chương trình

  • Chương trình làm gì: Liệt kê mọi xâu nhị phân độ dài n bằng bitmask.
  • Input: Nhập n.
  • Output: In tất cả cấu hình.
  • Biến chính: Biến: n, mask.
  • Luồng chạy: Đếm từ 0 đến 2^n - 1.
  • Độ phức tạp / kết quả kỳ vọng: O(n * 2^n).

Ví dụ 2: Sinh hoán vị bằng next_permutation

#include <iostream>
#include <algorithm>
using namespace std;

int main() {
    int a[4] = {1, 2, 3, 4};
    do {
        for (int i = 0; i < 4; i++) cout << a[i] << ' ';
        cout << endl;
    } while (next_permutation(a, a + 4));
    return 0;
}

Giải thích chương trình

  • Chương trình làm gì: Sinh các hoán vị của dãy 1..4.
  • Input: Không cần input.
  • Output: In từng hoán vị.
  • Biến chính: Biến: a.
  • Luồng chạy: Lặp next_permutation đến khi hết.
  • Độ phức tạp / kết quả kỳ vọng: O(n * n!).
