Bài này đi vào đồ thị có hướng: sắp xếp topo, thành phần liên thông mạnh và ý tưởng Tarjan qua mảng disc, low.

Trong bài này, điều quan trọng nhất là không chỉ nhớ định nghĩa mà còn phải biết bài toán nào nên dùng kỹ thuật nào, vì sao nó chạy nhanh hơn và khi nào không nên áp dụng. Khi đọc PDF gốc, hãy để ý các ví dụ và công thức vì chúng thường là điểm quyết định để làm bài tập.

Ở level của sinh viên năm nhất và người tự học, mục tiêu là hiểu thật rõ dữ liệu đang được lưu như thế nào, thao tác nào là O(1), thao tác nào là O(log n) hoặc O(n), rồi dùng kiến thức đó để chọn lời giải hợp lý hơn. Đây là tinh thần xuyên suốt toàn bộ khóa học.

Ví dụ minh họa

Ví dụ 1: Topological sort bằng Kahn

#include <iostream>
#include <vector>
#include <queue>
using namespace std;

int main() {
    int n = 4;
    vector<int> adj[5];
    adj[1] = {2, 3};
    adj[2] = {4};
    adj[3] = {4};
    vector<int> indeg(n + 1, 0);
    for (int u = 1; u <= n; u++) for (int v : adj[u]) indeg[v]++;
    queue<int> q;
    for (int i = 1; i <= n; i++) if (indeg[i] == 0) q.push(i);
    while (!q.empty()) {
        int u = q.front(); q.pop();
        cout << u << ' ';
        for (int v : adj[u]) if (--indeg[v] == 0) q.push(v);
    }
    return 0;
}

Giải thích chương trình

  • Chương trình làm gì: Sắp xếp topo bằng thuật toán Kahn.
  • Input: Không cần input.
  • Output: In một thứ tự topo.
  • Biến chính: Biến: adj, indeg, q.
  • Luồng chạy: Lấy dần đỉnh indegree 0.
  • Độ phức tạp / kết quả kỳ vọng: O(V + E).

Ví dụ 2: Tarjan cơ bản với disc và low

#include <iostream>
#include <algorithm>
#include <vector>
using namespace std;

vector<int> adj[1005];
bool vis[1005];
int disc[1005], low[1005], timer = 0;

void DFS(int u, int p = -1) {
    vis[u] = true;
    disc[u] = low[u] = ++timer;
    for (int v : adj[u]) {
        if (v == p) continue;
        if (!vis[v]) {
            DFS(v, u);
            low[u] = min(low[u], low[v]);
        } else {
            low[u] = min(low[u], disc[v]);
        }
    }
}

int main() {
    adj[1].push_back(2); adj[2].push_back(1);
    adj[2].push_back(3); adj[3].push_back(2);
    DFS(1);
    cout << disc[1] << ' ' << low[1] << endl;
    return 0;
}

Giải thích chương trình

  • Chương trình làm gì: Minh họa ý tưởng disc và low.
  • Input: Không cần input.
  • Output: In disc và low của đỉnh 1.
  • Biến chính: Biến: adj, vis, disc, low, timer.
  • Luồng chạy: DFS gán thời điểm thăm rồi cập nhật low.
  • Độ phức tạp / kết quả kỳ vọng: O(V + E).
