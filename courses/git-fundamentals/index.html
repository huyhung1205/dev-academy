Git Fundamentals

Khóa học nhập môn giúp bạn hiểu Git là gì, repository hoạt động ra sao, vì sao commit và branch quan trọng, cách merge thay đổi, và cách dùng GitHub để làm việc cùng người khác.

Bắt đầu học Trang chủ Danh sách khóa học

Đối tượng phù hợp

Người mới học Git, sinh viên, hoặc bất kỳ ai muốn quản lý mã nguồn rõ ràng hơn khi làm việc nhóm.

Mục tiêu học tập

  • Hiểu Git giải quyết vấn đề gì trong quản lý phiên bản.
  • Phân biệt repository, commit, branch và merge.
  • Biết quy trình làm việc cơ bản với GitHub.
  • Đọc được lịch sử thay đổi và chọn cách hợp nhất phù hợp.
