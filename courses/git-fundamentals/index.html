Git Fundamentals

Khóa học nhập môn dành cho sinh viên CNTT và người mới bắt đầu, đi từ version control, repository, commit, branch, merge đến GitHub, Pull Request, GitHub Pages và open source workflow.

Đối tượng phù hợp

Sinh viên CNTT và người mới chưa từng dùng Git, muốn học từ nền tảng đến workflow làm việc nhóm với GitHub.

Mục tiêu học tập

  • Hiểu Git, version control và sự khác nhau giữa Git với GitHub
  • Tự cài đặt, cấu hình và sử dụng Git trên Windows
  • Làm việc với repository local và remote bằng workflow chuẩn
  • Dùng branch, merge, pull request và GitHub Pages trong dự án thật
  • Tham gia các dự án open source cơ bản một cách an toàn

Tiên quyết

  • Biết dùng máy tính và trình duyệt web cơ bản
  • Không cần biết lập trình trước khi học
  • Có tài khoản GitHub để thực hành phần remote và Pull Request

Bài 1. Giới thiệu Git, cài đặt và repository

Hiểu Git là gì, cài Git trên Windows, cấu hình ban đầu và làm quen với repository local/remote.

Hiểu working directory, staging area, local repository và biết dùng git status, git add, git commit, git log, git diff, git show.
