Bài 1

Git là gì và Repository

Git giúp bạn lưu lại các mốc thay đổi của dự án, còn repository là nơi chứa toàn bộ lịch sử đó.

Tài liệu lý thuyết

Git là gì?

Git là hệ thống quản lý phiên bản phân tán. Nó cho phép bạn theo dõi thay đổi, xem lịch sử và quay lại mốc cũ khi cần.

Repository là gì?

Repository, hay repo, là nơi Git lưu toàn bộ lịch sử của dự án. Repo có thể nằm trên máy bạn hoặc trên GitHub.

Ví dụ khởi tạo repo

git init
git status
git add .
git commit -m "Initial commit"

Dòng lệnh trên tạo repository mới, kiểm tra trạng thái, thêm thay đổi và tạo commit đầu tiên.

Ví dụ minh họa

Khi làm một trang web nhỏ, mỗi lần hoàn thành một phần như header, footer hoặc CSS, bạn có thể tạo một commit để ghi lại mốc đó.
