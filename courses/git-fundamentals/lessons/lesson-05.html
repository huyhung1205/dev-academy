GitHub và quy trình cộng tác

GitHub là nơi lưu repository trên cloud để chia sẻ, theo dõi vấn đề, và phối hợp làm việc nhóm dễ dàng hơn.

Video mô phỏng

Video mô phỏng sẽ được cập nhật sau.

Tài liệu lý thuyết

Git và GitHub không giống nhau. Git là công cụ quản lý phiên bản trên máy bạn, còn GitHub là dịch vụ lưu trữ repository và hỗ trợ cộng tác.

Quy trình cơ bản khi làm việc với GitHub thường gồm clone repository, tạo branch, commit thay đổi, push lên remote, rồi tạo pull request nếu làm việc nhóm.

git clone https://github.com/user/project.git
git switch -c feature-header
git add .
git commit -m "Update header"
git push origin feature-header

Ví dụ minh họa

Bạn làm việc trong một nhóm 3 người: mỗi người tạo branch riêng, đẩy lên GitHub, sau đó review và merge lần lượt để tránh ghi đè công việc của nhau.
