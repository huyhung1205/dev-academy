# COURSE_TEMPLATE.md

## Standard Course

Each course must contain:

```txt
courses/<course-slug>/
├── index.html
└── lessons/
    ├── lesson-01.html
    ├── lesson-02.html
    ├── lesson-03.html
    ├── lesson-04.html
    └── lesson-05.html
```

## Course Overview Page

Required sections:

1. Course title
2. Course description
3. Target learners
4. Learning objectives
5. Lesson list
6. Start learning button
7. Back to homepage link

## Lesson Page

Required sections:

1. Video mô phỏng
2. Tài liệu lý thuyết
3. Ví dụ minh họa
4. Trắc nghiệm bằng JavaScript
5. Câu hỏi tự luận tự ôn tập
6. Đáp án tham khảo
7. Previous / Next lesson navigation
